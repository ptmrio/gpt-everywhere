Chat GPT3 Extension

First go to extension settings and enter an OpenAI API Key.

Press Ctrl + . anywhere.

Type a command in the command field, e.g. "Write an E-Mail reply"

Any selected text will autofill the context field.

If the source field from which you started the interation is an a textarea, textfield or any contenteditable field, the result will show in this element. Else it will be pasted into the context field.
